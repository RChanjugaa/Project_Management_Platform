# Role Permissions

## Administrator

- View administrator dashboard.
- Manage users and user status.
- Assign roles.
- Manage projects and system access.

## Project Manager

- View project manager dashboard.
- Create and manage projects.
- Assign team members to projects.
- Create and manage project tasks.
- Assign tasks to team members.

## Team Member

- View team member dashboard.
- View assigned projects.
- View assigned tasks.
- Update permitted task progress.
- Add task-related activity when permitted.

## Common Permissions

- Login.
- Logout.
- View authenticated profile.
